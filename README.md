# 🛒 FamilyCart - Smart Grocery Sharing App

Manage groceries together, effortlessly. **FamilyCart** เป็นแอปพลิเคชันช่วยจัดการรายการซื้อของในครอบครัวแบบ Real-time ที่ช่วยให้สมาชิกทุกคนในบ้านสามารถเพิ่ม ลบ และอัปเดตรายการสินค้าที่จะซื้อร่วมกันได้ง่ายๆ

## ✨ Features

- **Shared Family List:** สร้างกลุ่มครอบครัวหรือเข้าร่วมกลุ่มเดิมด้วย Access Code
- **Frequency Sorting:** แยกรายการตามความถี่ (One-time, Weekly, Monthly)
- **Smart Quantity Adjuster:** ปุ่มเพิ่ม/ลดจำนวนที่ใช้งานง่าย
- **Dark Mode Support:** รองรับการใช้งานทั้งโหมดสว่างและโหมดมืด
- **Real-time Sync:** เชื่อมต่อข้อมูลผ่าน Supabase มั่นใจได้ว่าข้อมูลอัปเดตเสมอ

## 🛠️ Tech Stack

- **Framework:** [Expo](https://expo.dev/) (React Native)
- **Database & Auth:** [Supabase](https://supabase.com/)
- **Navigation:** Expo Router (File-based routing)
- **Icons:** Expo Vector Icons (Ionicons)
- **Styling:** React Native StyleSheet (Custom Theme System)
