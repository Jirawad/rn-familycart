### ✨ Features
* **Shared List:** Share grocery items with family members via Access Code.
* **Quantity Control:** Easy plus/minus buttons for item units.
* **Frequency Filter:** Filter items by One-time, Weekly, or Monthly.
* **Dark Mode:** Supports both Light and Dark themes.

### 🛠 Tech Stack
* **Framework:** React Native (Expo)
* **Backend:** Supabase
* **Navigation:** Expo Router

### 🚀 How to Run
```bash
npm install
npx expo start
